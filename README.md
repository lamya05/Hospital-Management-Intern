 Hospital Management System
 ----------------------------
This is a Java based Hospital Management System project developed during my Java Development Internship at Landmine Soft.
The main goal of this project is to understand backend development concepts like REST APIs, database interaction, and layered architecture using Spring Boot.
---
Technologies Used
-------------------
- Java
- Spring Boot
- Spring Data JPA
- Maven
- MySQL
- GitHub
---
 Project Description
 --------------------
This application handles basic hospital related operations such as managing patients, doctors, and admin users.
The project is developed mainly to practice:
- Java fundamentals
- Spring Boot concepts
- REST API development
- Database connectivity
- Clean code structure
---
 Modules
 Patient
----------
- Patient registration
- Login functionality
- Stores patient personal details
- --
   Doctor
  -------
 - Doctor registration
- Login functionality
- Stores doctor information
- 
 Admin
--------
- Admin login
- View registered patients
- Pagination implemented for patient list
---
 Project Structure
 -----------------
The project follows a layered structure:
- Controller layer for handling requests
- Service layer for business logic
- Repository layer for database operations
- Entity and DTO classes for data handling
---
API Testing
-----------
- APIs are tested using Postman
---
Security Notes
------------------
- Passwords are stored in encrypted format
- Configuration files like `application.properties` are not pushed to GitHub for security reasons

---
How to Run the Project
-----------------------
1. Clone the repository
2. Create a database
3. Configure database details in `application.properties`
4. Run the application as a Spring Boot project
5. Test APIs using Postman
